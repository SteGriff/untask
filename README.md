# Untask

Just another ToDo list app

## Design goals

* Be small, light, fast
* Run on your Mum's feature phone and your Grandma's IE5
* Have an icon which does not feature a tick

## Tech

PHP5, Klein router, MySQL. Loosely MVC with DAL.
